# KNOWN_DIVERGENCES.txt
#
# Cases the cross-backend differential quarantines because a sibling lane is
# mid-fix. Each line quarantines one (case, backend) pair: for the tn_mps
# backend a quarantine means the driver SKIPS that case's tn leg entirely and
# counts it as quarantined -- never as a pass, never as a failure. Tolerances
# are NEVER loosened to make HEAD pass; this file is the only escape hatch, and
# a live entry is an OPEN bug that blocks the release (the ICC
# all_discovered_bugs_closed gate scans this file for non-comment lines).
#
# Format (whitespace-delimited; '#' comments and blank lines ignored):
#   <case_id>  <backend>  <owning_lane>  <free-form note incl. seed>
#
# case_id/seed reproduce byte-identically from the default corpus
#   scripts/gen_diff_corpus.py --out-dir DIR        (seed 0xB16B00B5)
# so any entry replays in isolation, at any profile.
#
# ---------------------------------------------------------------------------

# (n<=12, every family and depth), reproducing dense and the independent numpy
# reference to ~1e-13 with the tn leg armed at --tn-max-n 12.
# ---------------------------------------------------------------------------
# The three real bugs that formerly lived here are all fixed and regression-
# guarded, verified by a full empty-quarantine run (cross_backend_differential
