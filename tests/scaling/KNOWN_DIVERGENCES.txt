# KNOWN_DIVERGENCES.txt  (tests/scaling)
#
# Divergences this SCALING lane discovered, root-caused, and quarantines so the
# CI gate is stable while the owning lane fixes them. A quarantined divergence is
# REPORTED (with magnitude) but excluded from the exit status and from the
# scaling_differential_clean divergence count. Tolerances are NEVER loosened to
# make HEAD pass; this file is the only escape hatch. Full analysis + minimal
# repros: tests/scaling/FINDINGS.md.
#
# The quarantine here is RULE-BASED (encoded in the drivers), not a per-case
# list, because each finding has a structural trigger that spans a whole family
# of cases. The rules and their live-guard carve-outs:
#
# ===========================================================================
# FINDING S1 -- tn_mps adjacent two-qubit gate loses norm on a "bulk" bond
# ===========================================================================
# Owner lane : tn-lane   Suspect: src/algorithms/tensor_network/tn_gates.c
#                        (apply_gate_2q_adjacent, ~line 701)
#
# The adjacent 2q gate contracts the two site tensors, applies the 4x4 gate,
# SVD-splits, and renormalizes the singular values by their Frobenius norm --
# which is only the state norm when the MPS is in mixed-canonical gauge AT that
# bond. The gate path never establishes that gauge (every 2q gate ends with
# canonical = TN_CANONICAL_NONE). When the gate lands on a bond whose LEFT and
# RIGHT outer bonds are BOTH non-trivial (both neighbours already entangled),
# ||theta||_F != 1 and the rescale corrupts the norm.
#
# Minimal repro (4 qubits, forward, pure Clifford, all-adjacent):
#     H(0) CX(0,1) H(2) CX(2,3) CZ(1,2)
#   -> tn norm = 0.5, every probability halved (dense/reference are exact).
# Inserting tn_mps_mixed_canonicalize(state, 1) before the CZ makes dev -> 1e-16.
# Bond-dimension independent (chi = 8, 16, 64, 256, 1024 give identical error).
#
# This is almost certainly the SAME root cause the tn-lane tracks as the
# "n>=10 forward" divergence, but the TRUE envelope is n >= 4, pure Clifford,
# adjacent gates -- far wider than "n>=10 rotation circuits". Non-adjacent 2q
# gates reach it through the SWAP network (the SWAP makes the gate adjacent on a
# both-sides-entangled bond).
#
# QUARANTINE RULE (scaling_diff.c): a tn_mps divergence is KNOWN (excluded)
# unless the case is a LIVE tn guard on which the defect provably cannot fire:
#   - family ghz_chain  (single left-to-right CNOT sweep keeps the right outer
#     bond trivial at every gate -- no both-sides-entangled bulk bond forms);
#   - every case at n <= 4.
# A tn divergence on a live guard is a genuinely DIFFERENT bug and fails the gate.
# The absolute anchors -- dense-vs-reference and Clifford-tableau-vs-reference --
# are NEVER quarantined; they are exact at every n in this corpus.
#
# ===========================================================================
# FINDING S2 -- real-time TDVP is wrong for chains longer than 2 sites
# ===========================================================================
# Owner lane : tdvp-lane  Suspect: src/algorithms/tensor_network/tdvp.c
#                         (real-time environment / effective-H for interior sites)
#
# Real-time two-site TDVP reproduces exact dynamics for n <= 2 (a single bond),
# but is grossly wrong once there is an interior "bulk" site (n >= 3). For TFIM
# (J=1, h=1) from |0>^n, exact <Z_0>(0.4) = +0.71237, while TDVP gives:
#     n=2 -> +0.71237 (exact, norm 1.000)
#     n=3 -> +0.16857 (norm 0.990)
#     n=4 -> -0.11073 (norm 0.954)
#     n=5 -> +0.00085 (norm 0.921)
#     n=12 -> -0.42149 (norm 0.960)
# The error is dt-INDEPENDENT (0.02 and 0.002 identical -> not integrator order),
# bond-INDEPENDENT (bond 16..256 identical -> not truncation), and affects BOTH
# one-site and two-site variants. The reference is a dense Lanczos-Krylov
# exp(-iHt) verified against scipy.linalg.expm to 1e-6. DMRG -- which shares the
# same two-site SVD machinery -- reproduces ED to ~1e-6 at n=8,10,12, so this is
# a real-time-TDVP-specific defect, not the shared SVD split.
#
# Imaginary-time TDVP is a milder symptom of the same interior-site handling:
# two-site imag-time projection plateaus ~0.18 ABOVE the ED ground energy
# (still a valid variational value, so the hard bound E >= E0 holds and stays a
# LIVE check; the plateau itself is quarantined).
#
# QUARANTINE RULE (scaling_dmrg_tdvp.c): the TDVP real-time n>=3 <Z0>/norm checks
# and the imag-time "E ~ E0" convergence check are KNOWN (excluded). LIVE anchors
# that must still pass: DMRG-vs-ED (n=8,10,12), TDVP imag-time variational bound
# E >= E0, and TDVP real-time at n=2 (regression guard for the working case).
